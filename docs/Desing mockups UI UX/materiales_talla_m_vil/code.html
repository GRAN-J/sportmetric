Materiales

Verifica contar con todos los instrumentos necesarios antes de iniciar el protocolo de medición en campo o laboratorio.

Tallímetro

Instrumento de precisión utilizado para medir la estatura del sujeto. Debe contar con una base firme y un cursor deslizante perpendicular.

Escuadra antropométrica

Elemento esencial para garantizar un ángulo exacto de 90 grados al alinear la cabeza del sujeto en el plano de Frankfort.

Formato de registro

Documento estandarizado impreso para documentar sistemáticamente los resultados obtenidos durante la evaluación.

Planillero

Soporte rígido necesario para facilitar la escritura sobre el formato de registro en entornos donde no se dispone de mesa.
